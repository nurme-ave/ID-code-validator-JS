What's this?

The Estonian identification (ID) code is an unique 11-digit code and it has a set of requirements. This app checks all these requirements step by step and makes sure that the ID-code is indeed valid.

Type in your ID-code and hit 'Enter' - the result will be displayed to you underneath the form.
